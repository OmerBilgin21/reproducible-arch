# reproducible-arch

Personal Arch Linux dotfiles for a Hyprland + NVIDIA setup. Includes my development tooling alongside.

## Neovim Plugin Vendoring

I use Neovim as my primary editor and vendor all plugins directly in this repo under `config-shared/nvim/plugin-code/` for resilience — so my editor works even if I'm unable to connect to upstream plugin code.

I try to exclusively use plugins with MIT licence, however if you are a plugin author and would like your code removed from this repo, please open an issue or contact me and I will remove it immediately.
